# eslint-config-mo

> ESLint config

## Installation

```bash
$ npm i -D eslint eslint-config-mo
```

## Usage

Add `eslintConfig` into your package.json

```js
{
  "name": "my-awesome-project",
  "eslintConfig": {
    "extends": "mo"
  }
}
```

## Others

This is a fork of eslint-config-xo but with some custom rules
